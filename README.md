# WiFiFlashDrop
Телефон как беспроводная флешка для синхронизации обсидиана или просто как беспрводная флешка. Флешка. Беспроводная. А че бы нет?
Синхронизация syncthink мне не зашла категорически. Мне надо чтобы я точно указал проге - это хранилище главное, его и копируй. Чтобы нажимать меньше кнопок в WinSCP сделана эта утилита.


Для работы требуется приложение на андроид (я использую бесплатную SSH/SFTP Server (от Banana Studio).
Также потребуется через WinSCP подрубится к своему телефону-серверу. А эта утилита чисто батник с простым интерфейсом, который превращат дальнейшую синхронизацию как: запустил-кликнул-вышел.

Кнопки синхронизации:

⬆ Синхронизировать: ПК → Телефон: Делает хранилище на ПК главным. Копирует все новые и измененные заметки на телефон.
⬇ Синхронизировать: Телефон → ПК: Делает хранилище на телефоне главным. Копирует заметки, которые вы редактировали в дороге, на компьютер.

👀 Предпросмотр (ПК → Телефон и Телефон → ПК): Отдельные кнопки. Программа проверит изменения и покажет лог, но не будет реально копировать или удалять файлы. Может пригодиться для проверки перед запуском. Вынесено в отдельные кнопки, чтобы случайно не забыть снять галочку.

Опции синхронизации (галочка):

Разрешить удаление (Зеркалирование):

Выключено (по умолчанию): Безопасный режим. Если вы удалили файл на ПК, на телефоне он останется. Происходит только копирование и обновление.
Включено: Режим точной копии (Mirror). Если файла нет в главной папке, он будет удален в папке назначения.


Сборка exe:
```
py -m PyInstaller --onefile --windowed "WiFiFlashDrop.py"
```
