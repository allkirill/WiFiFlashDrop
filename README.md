# Obsidian-Sync
Телефон как беспроводная флешка для синхронизации обсидиана
Синхронизация syncthink мне не зашла категорически. Мне надо чтобы я точно указал проге - это хранилище главное, его и копируй. Чтобы нажимать меньше кнопок в WinSCP сделана эта утилита.


Для работы требуется приложение на андроид (я использую бесплатную SSH/SFTP Server (от Banana Studio).
Также потребуется через WinSCP подрубится к своему телефону-серверу. А эта утилита чисто батник с простым интерфейсом, который превращат дальнейшу синхронизацию как запустил кликнул и вышел.

⬆ ПК → Телефон: Делает хранилище на ПК главным. Копирует все новые и измененные заметки на телефон.
⬇ Телефон → ПК: Делает хранилище на телефоне главным. Копирует заметки, которые вы редактировали в дороге, на компьютер.

Опции синхронизации (галочки)
Разрешить удаление (Зеркалирование):
Выключено (по умолчанию): Безопасный режим. Если вы удалили файл на ПК, на телефоне он останется. Происходит только копирование и обновление.
Включено: Режим точной копии (Mirror). Если файла нет в главной папке, он будет удален в папке назначения.
Только предпросмотр: Программа проверит изменения и покажет лог в консоли, но не будет реально копировать или удалять файлы. Может пригодится для проверки перед запуском.
